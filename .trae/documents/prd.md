# 旭日扬光光伏网站重建 - 产品需求文档 (PRD)

## 1. 产品概述

南京旭日扬光光伏科技有限公司官网重建项目，旨在将原网站 (http://www.sunrising-solar.com/) 完整迁移并重建为一个现代化、响应式的独立网站。保留原网站全部文本、图片、布局结构和交互功能，同时优化代码结构、提升加载性能并确保多设备适配。

- 目标用户：潜在客户、合作伙伴、行业人士，关注光伏发电系统解决方案的企业和个人
- 核心价值：展示公司产品与服务、工程案例实力、建立品牌信任感

## 2. 核心功能

### 2.1 功能模块

1. **首页**：轮播横幅、产品分类展示、公司简介摘要、工程案例展示、新闻公告
2. **公司简介页**：公司介绍、主营业务、企业愿景
3. **产品中心页**：产品分类筛选、产品列表展示、产品详情
4. **工程案例页**：项目案例列表、案例详情
5. **联系我们页**：联系信息、在线留言表单

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝色 (#1a365d) 代表科技与专业，搭配太阳能橙色 (#f6ad55) 作为强调色
- 辅助色：浅灰背景 (#f7fafc)、白色卡片、深灰文字 (#2d3748)
- 按钮风格：圆角按钮，hover时带阴影提升效果
- 字体：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular
- 布局风格：顶部固定导航、卡片式内容布局、全宽轮播横幅
- 图标风格：使用 lucide-react 线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 首页 | 轮播横幅 | 全宽大图轮播，渐变遮罩文字，自动切换3-5秒 |
| 首页 | 产品分类导航 | 6个圆形图标+文字，网格排列，hover放大效果 |
| 首页 | 产品推荐 | 3列卡片网格，图片+标题+简介，hover阴影 |
| 首页 | 公司简介 | 左图右文布局，MORE+链接按钮 |
| 首页 | 工程案例 | 横向滚动卡片，图片+标题+日期 |
| 公司简介 | 公司介绍 | 居中大段文字，配装饰线条 |
| 公司简介 | 主营业务 | 4列图标+文字卡片 |
| 公司简介 | 企业愿景 | 引用样式，配背景图 |
| 产品中心 | 分类标签 | 横向标签栏，选中态高亮 |
| 产品中心 | 产品网格 | 响应式网格，3/2/1列自适应 |
| 工程案例 | 案例列表 | 卡片网格，含日期标签 |
| 联系我们 | 联系信息 | 图标+文字列表 |
| 联系我们 | 留言表单 | 输入框+文本域+提交按钮 |

### 4.3 响应式设计

- 桌面优先设计，适配1280px+屏幕
- 平板适配（768px-1279px）：导航折叠为汉堡菜单，网格从3列变2列
- 移动端适配（<768px）：单列布局，导航抽屉，卡片全宽
- 触摸优化：按钮最小44px触摸区域，滑动轮播支持手势
